About This Course

Welcome to WDD 131 - Dynamic Web Fundamentals. In this course, I'll be learning essential web development skills including HTML, CSS, and JavaScript. The assignments will focus on creating responsive, accessible, and interactive web pages while following industry best practices.

Throughout the semester, I'll be applying design principles, implementing modern layouts, and developing dynamic content using JavaScript to create a comprehensive web experience.

Course Goals

My goals for this course include mastering JavaScript fundamentals, creating responsive layouts that work across devices, and developing clean, semantic HTML with proper CSS implementation.

I'm particularly interested in learning how to use JavaScript to add interactivity to web pages and how to implement modern design principles to create appealing user interfaces.
